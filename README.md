<h1 align="center">CRUD em Typescritpt</h1>

## Descrição
Simples CRUD de produtos fazendo suas funções básicas.

## Requisitos
Ter o node e o postgresql instalado

## Ferramentas 
<li>nodejs </li>
<li>typescript </li>
<li>typeORM </li>
<li>postgresql </li>

## Modo de utilização
1. Instale as dependências
> npm install

2. Inicialize a aplicação:
> npm run build

> npm run typeorm migrate:run

> npm start
