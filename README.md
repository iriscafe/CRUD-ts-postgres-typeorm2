<h1 align="center">CRUD em Typescritpt</h1>

## Descrição
Simples CRUD de produtos fazendo suas funções básicas.

## Ferramentas
<li>nodejs </li>
<li>typescript </li>
<li>typeORM </li>
<li>postgresql </li>

## Modo de utilização
1. Rode o comando para baixar as dependências utilizando docker:
> run docker-compose

2. Inicialize a aplicação:
> npm run build
> npm start
